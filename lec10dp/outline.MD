# 动态规划 Dynamic Programming

##递归/分治算法的缺点（重复子问题求解）
- 重复子问题会严重降低算法效率

### 例子
- fib(n)
- n!

##动态规划的基本思想
- 利用存储来避免重复子问题的求解

### 打表方法（静态规划）
- 自底向上计算（递推代替递归)


### 动态规划
- 自顶向下递归
- 自下而上计算
- 利用查表来避免重复子问题的求解
